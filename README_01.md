# Piscine_Php

## Installation
`make && make clean` | Compiles and creates the library `libftprintf.a`.

`make re && make clean` | Recompiles.

## Usage
`gcc resources/test_main.c libftprintf.a && ./a.out` | Prints the test main provided as example.
